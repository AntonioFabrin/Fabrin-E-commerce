# Fabrin E-commerce 🛒

## 💻 Sobre o Projeto
O Fabrin E-commerce é uma aplicação completa de comércio eletrônico. Este projeto foi desenvolvido do zero com o objetivo de construir uma plataforma robusta, focando nas melhores práticas de desenvolvimento, tipagem estática e gerenciamento eficiente de banco de dados. 

## 🚀 Tecnologias Utilizadas
O projeto está sendo desenvolvido com as seguintes tecnologias:

* **Frontend/Mobile:** React Native
* **Linguagem:** TypeScript
* **Requisições HTTP:** Axios
* **Banco de Dados:** MySQL 
* **Gerenciamento de BD:** HeidiSQL

## 🛠️ Como rodar o projeto localmente

### Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/) e o banco de dados configurado no HeidiSQL.

### Rodando a aplicação
```bash
# Clone este repositório
$ git clone [https://github.com/AntonioFabrin/Fabrin-E-Commerce.git](https://github.com/AntonioFabrin/Fabrin-E-Commerce.git)

# Acesse a pasta do projeto no terminal/cmd
$ cd nome-do-repositorio

# Instale as dependências
$ npm install 
# ou yarn install

# Execute a aplicação
$ npm start
# ou yarn start
